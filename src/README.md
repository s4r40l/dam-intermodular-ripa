Este directorio contiene **todo el código fuente de la aplicación RIPA**, desarrollada en Java utilizando Maven, JavaFX y JDBC para la gestión de productos, supermercados y registros de precios.

La estructura sigue el estándar Maven:

```
src/
 ├── main/
 │    ├── java/
 │    │    ├── controller/        → Controladores de lógica de negocio
 │    │    ├── DAO/               → Acceso a datos (JDBC)
 │    │    ├── database/          → Conexión MySQL (ConexionBD)
 │    │    ├── model/             → Clases del modelo de datos
 │    │    ├── xml/               → Importación y exportación XML
 │    │    └── Main.java          → Punto de entrada de la aplicación
 │    └── resources/              → Recursos del proyecto (FXML, etc.)
 │
 └── test/
      └── java/
           ├── controller/        → Tests de controladores
           └── DAO/               → Tests de DAOs
```
