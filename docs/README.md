# Documentación del Proyecto — Carpeta `docs`

Este directorio contiene **toda la documentación oficial del Proyecto Intermodular de 1º DAM**, organizada por módulos.  
Cada subcarpeta incluye los informes, memorias y archivos requeridos por cada asignatura.

## 📁 Contenido

- **/bbdd/** — Documentación del módulo de Bases de Datos  
- **/entornos/** — Informe técnico del módulo de Entornos de Desarrollo  
- **/lenguajesDeMarca/** — Documentación y archivos del módulo de Lenguajes de Marca  
- **/programacion/** — Documentación del módulo de Programación  
  - Incluye un **ZIP con todo el código fuente completo**
- **/sistemas/** — Informe técnico del módulo de Sistemas Informáticos  
- **/mpo/** — Documento de mejoras MPO aplicadas al proyecto
